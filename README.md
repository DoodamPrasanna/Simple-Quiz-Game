# 🧠 Quiz Website

A simple and interactive Quiz Website built using **HTML**, **CSS**, and **JavaScript**. This project allows users to answer multiple-choice questions, receive instant feedback, and view their final score at the end of the quiz.

## 🚀 Features

* Multiple-choice quiz questions
* Interactive and user-friendly interface
* Real-time answer checking
* Score calculation and display
* Responsive design for desktop and mobile devices
* Clean and modern UI

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

## 📂 Project Structure

```text
Quiz-Website/
│
├── index.html
├── style.css
├── script.js
└── README.md
```

## 🎯 How to Run the Project

1. Clone the repository:

```bash
git clone https://github.com/your-username/quiz-website.git
```

2. Open the project folder.

3. Double-click `index.html` or open it using Live Server in VS Code.

4. Start answering the quiz questions and test your knowledge!

## 📸 Preview

The website presents questions one at a time with multiple answer options. Users can select an answer, move to the next question, and view their final score after completing the quiz.
